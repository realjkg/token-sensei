How it works
  • MediaRecorder captures 8 seconds max
  • WAV blob → POST /voice/verify
  • Resemblyzer speaker embedding + MFCC similarity scored server-side

useVoiceRecorder hook
  • MediaRecorder API — WAV via mimeType detection
  • 8-second auto-stop with visual countdown
  • Level (0–1) drives bar heights via getByteFrequencyData
  • Audio stays local — no streaming to the cloud
Air-gap note: The WAV blob travels only to the local FastAPI server on the LAN. Resemblyzer runs on-device (CPU). No external API call.

Identity confirmed

When Resemblyzer scores ≥ 0.72 (high confidence), Emma sees a green confirmation card with the confidence percentage. One tap starts her session.

Three result levels
  • High (≥0.72) — confirmed, green card
  • Medium (≥0.50) — uncertain, retry offered
  • Low (<0.50) — not matched, 3 attempts then parent override
No match path: after 3 failures, parent enters their password on the same device to approve the session — no separate device needed.
